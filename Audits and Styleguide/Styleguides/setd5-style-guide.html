Page Layer System

The homepage uses a five-layer background system that creates depth from dark to light as the user scrolls down.

Interior pages use a simpler system: Nav (#214F64) → Interior hero (#EFEEEA, light text removed) → Page body (#F5F4F1) → Footer (#2A627A).

:root {
  --indigo:      #9e9170;  /* nav, headings, primary buttons */
  --indigo-mid:  #a7c4ba;  /* links, labels, H3 */
  --gold:        #C4952A;  /* accent, outline buttons */
  --gold-hover:  #D4B84C;  /* gold hover state */
  --cream:       #F5F4F1;  /* page background */
  --parchment:   #f0ede7;  /* cards, surfaces */
  --parchment-d: #e0dcd5;  /* borders, dividers */
  --ink:         #090806;  /* body text */
  --ink-muted:   #4a4642;  /* captions, secondary text */

  --sage-lt:     #e8e6e0;  /* callout / info box background */

  --font-body:    'DM Sans', system-ui, sans-serif;  /* single font — all headings and body */
}

Typography

DM Sans is used for everything — headings and body text. Single font, single Google Fonts load. Interior page hard cap: 20px maximum for any heading, 16px maximum for body text. Hero headers on interior pages are exempt from the cap.

Do & Don't

The rules that keep the design coherent. These are derived from real decisions made on the site.

Typography
Do
  • Use DM Sans for everything — headings and body
  • Set interior H2s to exactly 18px / #a7c4ba
  • Add 1rem (16px) margin below card titles
  • Keep body text at 16px maximum
  • Use font-weight 600 for all headings
Don't
  • Use Poppins — it was removed from the system
  • Exceed 18px for interior page headings
  • Use font-weight 700 for body text or labels
  • Mix heading colors — all H2s are #a7c4ba, all H3s are #9e9170
  • Set body text above 16px on interior pages
Color
Do
  • Use only the nine CSS variables in the palette
  • Use #e8e6e0 (--sage-lt) for info and callout boxes
  • Keep white (#fff) for content card backgrounds
  • Use --cream (#F5F4F1) as the default page background
Don't
  • Invent new hex values outside the palette
  • Use a light background on footer elements — footer is always dark indigo
  • Put dark-colored text on the dark footer without rgba opacity
  • Use gold for body text or long-form reading
Buttons & actions
Do
  • Limit primary buttons to one per section
  • Pair a secondary/ghost button with a primary
  • Use the gold nav CTA exclusively in the nav bar
Don't
  • Place two primary buttons side by side
  • Use the nav CTA style as a page-level button
  • Change button border-radius — always 6px for page buttons, 4px for nav

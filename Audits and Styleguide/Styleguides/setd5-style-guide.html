CSS Variables — paste at the top of your stylesheet
/* ── The Counsel Palette ───────────────────── */
:root {
  --indigo:      #1E3A4F;  /* nav, headings, primary buttons */
  --indigo-mid:  #456A7C;  /* links, labels, H3 */
  --gold:        #9E7E42;  /* accent, outline buttons */
  --gold-hover:  #C4A362;  /* gold hover state */
  --cream:       #F5F4F1;  /* page background */
  --parchment:   #E8E6DF;  /* cards, surfaces */
  --parchment-d: #D0CCC2;  /* borders, dividers */
  --ink:         #1C1B18;  /* body text */
  --ink-muted:   #5A5850;  /* captions, secondary text */

  --font-heading: 'DM Sans', system-ui, sans-serif;  /* H1–H4, logo, stat numbers */
  --font-body:    'Poppins', system-ui, sans-serif;  /* paragraphs, nav, buttons, labels */
}

Typography

DM Sans for headings, Poppins for body text. Both load from a single Google Fonts link. Use the CSS variables --font-heading and --font-body throughout.

Google Fonts — paste inside <head>

Buttons

Three styles. One primary maximum per section. Never two primary buttons side by side.

Primary — one per section maximum
Secondary / outline — pairs with primary
Ghost / tertiary — low-emphasis

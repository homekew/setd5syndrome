Rule: eyebrow / label text on dark cards. When a section label or uppercase eyebrow sits inside a dark navy card (#1E3A4F or #1a2744), never use color: var(--teal) / #456A7C — steel indigo on midnight indigo has near-zero contrast. Use #9FBBC8 (steel tint) or #a8cfc8 (soft teal) instead. This applies to any element with color:#2a7068 or color: var(--teal) that lives inside a dark background container — these need the light override.

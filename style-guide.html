Design Principles

Warmth & Approachability

The palette combines warm brown with vibrant blue-grey to feel welcoming and family-focused rather than institutional or corporate. This supports the "built by a parent, for families" positioning.

Clear Visual Hierarchy

Color guides the eye intentionally: charcoal for primary content, vibrant blue-grey for emphasis and titles, warm brown for navigation and primary actions. This creates predictable, scannable layouts.

Accessibility First

All color combinations maintain WCAG AA contrast ratios. The vibrant blue-grey (#4f8f9e) provides strong contrast against both cream and charcoal backgrounds.

Consistency & Simplicity

Only five colors are used site-wide. This creates a cohesive, unified visual system that feels intentional and professional.

Distinctness

This palette is visually distinct from the SETD5 Syndrome Foundation brand (which uses teal, coral, and gold), reinforcing that this is a separate parent resource site.

Usage Guidelines

When to Use Each Color

Color Hex Primary Uses Avoid
Charcoal #383228 Body text, paragraphs, general content, dark text on light backgrounds As accent color; backgrounds
Cream #F5F4F1 Page background, card backgrounds, light surfaces As text color (insufficient contrast)
Warm Brown #565246 Navigation bar, primary buttons, top card borders, primary CTAs Body text on light backgrounds
Vibrant Blue-Grey #4f8f9e Card titles, section headings, emphasis text, borders, callout labels, badges Large text backgrounds

Color Combinations to Avoid

  • Cream text on cream backgrounds (no contrast)
  • Blue-grey as large background area (too much visual weight)
  • Brown text on cream for body copy (insufficient contrast)
  • Mixing with colors outside the defined palette
